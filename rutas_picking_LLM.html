Optimización de rutas de picking con IA

Frente a rutas fijas, la IA calcula la secuencia óptima de recogida considerando ubicaciones actuales, obstáculos en tiempo real y otras carretillas en movimiento. Este ejemplo conecta con la API de Claude para mostrar el razonamiento real del LLM.

Configuración — API key de Anthropic

Para que el LLM responda en tiempo real necesitas una API key de Anthropic. Puedes obtenerla en console.anthropic.com → API Keys → Create Key. Las primeras llamadas tienen crédito gratuito. La key se guarda solo en tu navegador, nunca se envía a ningún servidor propio.

Decisiones tomadas por el LLM — con justificación
    ¿Qué ocurre cuando se despeja el bloqueo?
    El sistema vuelve a consultar al LLM con el nuevo estado del almacén. La ruta se recalcula en segundos — no hay que reprogramar nada. Esto es lo que diferencia la optimización en tiempo real de una ruta fija programada.
